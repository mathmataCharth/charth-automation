# Charth Automações e Ferramentas

Repositório de ferramentas internas, automações e utilitários desenvolvidos para suportar operações, dados e integrações da Charth.

O objetivo é centralizar scripts e pequenos projetos que auxiliam em:

* auditorias e validações de dados
* rotinas operacionais
* integrações entre sistemas
* análises e suporte à tomada de decisão

Cada ferramenta é independente, com sua própria estrutura, dependências e documentação.

## Estrutura

```
/tools
  /conferencia-produto-estore
  /<outras-ferramentas>
```

## Ferramentas

### conferencia-produto-estore

Ferramenta para conferência e validação de produtos relacionados ao processo de estore, com foco em identificar inconsistências e garantir integridade dos dados.

## Princípios

* Baixo acoplamento entre ferramentas
* Execução independente
* Organização simples e escalável
* Foco em problemas operacionais reais
